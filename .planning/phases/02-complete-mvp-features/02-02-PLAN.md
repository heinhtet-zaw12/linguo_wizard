Define a class `ScoreData` with these fields:

- `final int grammarScore;` (0-100)
- `final int vocabularyScore;` (0-100)
- `final List<GrammarCorrection> grammarCorrections;`
- `final int xpEarned;`

Define a helper class `GrammarCorrection` in the same file:
- `final String original;`
- `final String corrected;`
- `final String explanation;`
- Const constructor with required fields

Add a `factory ScoreData.fromJson(Map<String, dynamic> json)` that:
- Reads `overallScore`, `fluencyScore`, `grammarScore`, `vocabularyScore` as int (with fallback to 0)
- Reads `grammarCorrections` as a List, mapping each to GrammarCorrection
- Defaults xpEarned to 10 (flat rate per D-11 scope — configurable later)

Add `factory ScoreData.fallback()` that returns all scores 0, empty corrections, xpEarned 0 (used when evaluation fails).

**Step B: Add evaluation prompt to AppConfig** in `lib/core/config/app_config.dart`:

Add a static const string `evaluationPromptTemplate` that is a multi-line string:
```
You are an English language teacher evaluating a student's conversation performance.

The student's conversation goal was: {goal}

Analyze the conversation transcript below and provide:
1. An overall score (0-100) based on how well the student achieved the goal
2. A fluency score (0-100) based on natural flow and coherence
3. A grammar score (0-100) based on grammatical accuracy
4. A vocabulary score (0-100) based on word choice and range
5. A list of grammar corrections with original text, corrected text, and explanation

Be fair but encouraging. Score generously for beginners (A1-A2) and stricter for advanced (B1+).

Conversation transcript:
{transcript}
```

Add `static const int xpPerScenario = 10;` (per research recommendation — flat 10 XP).

**Step C: Create EvaluationService** at `lib/core/services/evaluation_service.dart`:

Create class `EvaluationService` following the AiService pattern (plain Dart class, no Riverpod):

```dart
class EvaluationService {
  Future<ScoreData> evaluateGoal({
    required String scenarioGoal,
    required String transcript,
  }) async { ... }
```

Implementation:
1. Load API key from `AppConfig.geminiApiKey` — throw StateError if empty (same pattern as AiService)
2. Create a `GenerativeModel` with:
   - `model: AppConfig.geminiModel`
   - `apiKey: AppConfig.geminiApiKey`
   - NO system instruction (this is a one-shot evaluation, not a chat)
3. Build the prompt by replacing `{goal}` and `{transcript}` in `AppConfig.evaluationPromptTemplate`
4. Call `model.generateContent()` with:
   - `[Content.text(prompt)]`
   - `generationConfig: GenerationConfig(temperature: 0.3, responseMimeType: 'application/json', responseSchema: ...)` where the schema matches the ScoreData JSON structure (overallScore, fluencyScore, grammarScore, vocabularyScore as integer, grammarCorrections as array of objects with original/corrected/explanation strings)
5. Parse the response: `jsonDecode(response.text!)` and pass to `ScoreData.fromJson()`
6. Wrap entire method in try-catch: on any exception, log and return `ScoreData.fallback()`
7. Add the Gemini `Schema` object for the response schema matching the ScoreData structure (use `SchemaType.object` with properties for each field)

Import `dart:convert` for jsonDecode, `google_generative_ai` for GenerativeModel/Schema, and the ScoreData model.
